Later Dude Widget for Mac OS X.

Later Dude provides a convenient way for you to remember URLs, files, folders and pretty much anything you can drag and drop on Mac OS X. You just start dragging your item of choice, fire up the dashboard, and drop it like it's hot on your Later Dude widget. Later Dude stores a reference to the item for later use. You can click on the item to open it with its default application (web browser for URLs, Finder for folders, etc.) or drag it back out, say to drop a file onto a file chooser dialogue to save yourself a few clicks. Note that Later Dude only keeps the URI of an item, not the item itself, so make sure not to delete files or folders you want to reference later.

Installation

Mac OS X v10.4 Tiger is required. If you're using Safari, click the download link. When the widget download is complete, the widget installer appears. Click Install if you want the widget installed on your Mac. If you're using a browser other than Safari, click the download link. When the widget download is complete, unarchive and open it to show the widget installer.
