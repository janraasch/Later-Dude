Later Dude

Your Friendly URL and File Dropbox on the OS X Dashboard.

Download this project as a .zip file

Later Dude Widget for Mac OS X.

Later Dude provides a convenient way for you to remember URLs, files, folders and pretty much anything you can drag and drop on Mac OS X.

  • Start dragging your item of choice, fire up the dashboard, and drop it like it's hot on your Later Dude widget. Later Dude stores a reference to the item for later use.
  • Click on the item to open it with its default application (web browser for URLs, Finder for folders, etc.) or drag it back out, say to drop a file onto a file chooser dialogue to save yourself a few clicks.

Dropping an application onto Later Dude turns the widget into a handy application launcher. The possibilities are endless, but most importantly you can drop your dearest Party Rock Anthem mp3 file on there and have that available to you at all times, just like you should.

Note that Later Dude only keeps the URI of an item, not the item itself, so make sure not to delete files or folders you want to reference later.

Installation

Mac OS X v10.4 Tiger is required.

  • Safari:
    • Click the download link.
    • When the widget download is complete, the widget installer appears.
    • Click Install if you want the widget installed on your Mac.
  • If you're using a browser other than Safari:
    • Click the download link.
    • When the widget download is complete, unarchive and
    • Open it to show the widget installer.
